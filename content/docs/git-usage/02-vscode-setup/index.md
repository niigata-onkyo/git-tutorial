このページでは、VSCode で Git を利用する準備について記載します。

## 1. はじめに

はじめに、基本的には Git がインストールされていれば、標準の VSCode で Git を利用することができます。

そのため、この資料で記載するのは「VSCode での Git 利用をより円滑にするもの」になります。

## 2. 拡張機能のインストール
