このページでは、GitHub アカウントの設定についていくつか紹介します。

(ちなみに、GitHub は**日本語設定がありません** (???)[^lang])

## 1. GitHub のトップページ

GitHub のトップページを開きます。

<https://github.com>

こちらの画面が表示されます。

![GitHub Home](01.png)

## 2. 設定ページを開く

右上のアイコンをタップします。

![GitHub Home → Settings](02.png)

サイドバーが開くので、Settings を開きます。

![GitHub サイドバー](03.png)

## 3. 名前・アイコン設定

表示された部分で名前とアイコンを設定できます。わかりやすい名前に変えておきましょう。

![GitHub 名前・アイコン](04.png)
