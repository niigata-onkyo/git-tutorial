# 音響 Git チュートリアル

[![Deploy Hugo site to Pages](https://github.com/niigata-onkyo/git-tutorial/actions/workflows/pages.yaml/badge.svg?branch=main)](https://github.com/niigata-onkyo/git-tutorial/actions/workflows/pages.yaml)

<https://niigata-onkyo.github.io/git-tutorial/>

**音響 Git チュートリアル** は、音響工学プロジェクトにて、Git 利用を進める際の補助資料として用意しました。

ベースの作成及び、主な保守は [@mint73](https://github.com/mint73) が担当しています。

## 開発

本資料の作成では、Hugo の Hextra テンプレートを利用しています。

Hugo, Go, Git をインストールした上で、以下のコマンドを実行することでローカル開発ができます。

```sh
hugo server
# localhost:1313
```

詳細は[開発資料](./docs/DEVELOPMENT.md) (英語) を参照してください。
